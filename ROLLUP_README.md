# DiscSharp Documentation Pass

This rollup starts from the green DiscSharp build baseline plus the Discord API alignment pass and adds a complete documentation set for the current product surface.

## What this documentation pass adds

- Product positioning without empty bragging.
- A clear explanation of why DiscSharp is taking a different direction from common Discord.Net-style usage.
- Getting-started wiring for the current solution.
- Autofac-first composition guide.
- REST API usage guide.
- Gateway orchestration API guide.
- Interaction pipeline guide.
- Components and modals guide.
- Observability guide.
- Testing guide.
- Raid Manager and Music Player examples.
- Current status and roadmap.

## Main entry point

Open `docs/README.md`.

## Build note

This pass changes Markdown documentation and manifest files only. It should not affect C# build output.
